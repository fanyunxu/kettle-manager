#### 软件架构

#### 使用说明
开发项目需要配置maven镜像
```xml
<mirror>
        <id>pentaho</id>
        <mirrorOf>*</mirrorOf>
        <name>Nexus pentaho</name>
        <url>http://nexus.pentaho.org/content/groups/omni/</url>
</mirror> 
```
